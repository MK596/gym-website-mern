# T2K Gym - MERN Stack Website

A premium informational website for T2K Gym built with the MERN stack (MongoDB, Express, React, Node.js).

## Features
- **Premium UI**: Modern dark mode with neon branding.
- **Service Listings**: Detailed workout programs and professional trainers.
- **Membership Informational**: Detailed tiers and pricing structures.
- **Public Inquiries**: Direct contact form for potential members.

## Prerequisites
- Node.js installed.
- MongoDB Connection String (in `backend/.env`).

## Setup & Run

### 1. Backend
Open a terminal in the root directory:
```bash
cd backend
npm install
npm run dev
```
*Server runs on http://localhost:5000*

### 2. Frontend
Open a **new** terminal in the root directory:
```bash
cd frontend
npm install
npm run dev
```
*Client runs on http://localhost:5173*
